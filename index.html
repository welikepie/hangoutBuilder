HangoutBuilder

Welcome to the HangoutBuilder project. The HangoutBuilder application allows you to create your own Google+ Hangouts. Below, you can download both Windows and Mac versions of the application, as well as a guide on how to use the HangoutBuilder and create your own Google+ Hangouts.

View Demo Hangout

Application

To create your own Google+ Hangouts you must use the HangoutBuilder.app (Mac Version) or HangoutBuilder.exe (Windows Version) to create the Hangout files you'll need to use when creating your Google+ Hangout.

To download the application, please select either the Mac or Windows version of the application, below.
